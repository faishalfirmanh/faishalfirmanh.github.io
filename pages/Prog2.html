Event pada javascript

Bagi para developer web, javascript merupakan hal yang tidak bisa bisa dipisahkan terutama web desain, dengan js (javascript) dapat membuat tampilan web menjadi lebih interaktif

Salah satu yang saya suka dari js adalah dapat memainkan animasi,tulisan ini akan membahas mengenai event, event marupakan interaksi js dengan html ketika user melakukan manupulasi pada halaman atau memodif

berikut beberapa daftar event js, yang dapat membuatmu memodif halaman web

Nama event Penjelasan Contoh script
offline() digunakan untuk mengecek apabila terhubung ke internet atau tidak klik
onafterprint() posisinya di tag body ketika dokument html di print(ctrl + p) akan memunculkan alert, bahwa dokument sedang diprint klik
onbeforeprint() ketika dokument html ctrl+p sebelum di print akan memunculkan alert klik
onbeforeload() sebelum halaman di reload atau f5, akan menampilkan alert, atau saat di klik link klik
oncanplay() dan oncanplaythrough() kedua event inihampir sama penepatan di tag video oncanplay="namaFungsi",ketika menambahkan event ini dihtmk video dapat di paus, volume, dll seperti media player biasahnya klik
onchange() akan memberitau ketika text atau tindakan lain,ketika dirubah dengan muncul aret sesua yng kita mau klik
onscroll() mendeteksi jumlah scroll, uji coba di tag textarea klik
onmouseover() ketika kursor mouse diletakkan di obyek, maka obyek dapat terlihat berubah atau sesuai dengan yang kita inginkan klik
onmouseout() ketika kursor mouse keluar dari obyek, maka obyek dapat terlihat berubah atau tetap sesuai dengan yang kita inginkan klik
