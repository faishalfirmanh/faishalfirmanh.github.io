Query join mysql lebih dari 2 tabel

Database mysqli merupakan Database yang sering digunakan oleh para pengembang, baik yang masih pemula, maupun dalam bentuk real projek, mengenai macam macam query mysql yang akan dibahas disini adalah JOIN

join memiliki 3 macam inner, right dan left, pada tulisan ini akan dibahas mengenai inner join,sebelumnya di bawah ini gambar perbedaannya

  1. langah pertama buatlah sebuah tabel dengan contoh gambar1, atau copas script dibawah ini, haru satu satu


  2. masukkan data tabel pengujung seperti dibawah ini

  3. isi tabel jenis

  4. buat relasi di database, pada tabel tiket->structure->relation view

    pertama hubungkan 1 tabel kolom dulu, lalu save, dilanjutkan menghungkan kolom berikutnya

  5. ketika seluruh kolom sudah terhubung, lalu inputkan data pada tabel tiket

    6. ketika insert pada kolom pengunjung dan jenis tiket, otomatis akan muncul keterangan dan idnya, tapi yang disimpan nanti berupa id saja.

    setelah data ditambahkan, maka yang disimpan adalah id dari tabel utama tblpengujung dan tbljenistiket

  6. lalu bagaimana cara menampilakn nama pengujung dan jenis tiketsedangkan yang disimpan saja berupa id, ini guna query join

gunakan qury dibawah ini

SELECT tiket.idTiket, pengunjung.nama as 'nama pemesan', jenistiket.namaTiket as 'kategori tiket', tujuan FROM tiket JOIN pengunjung on tiket.idPengunjung = pengunjung.idPengunjung JOIN jenistiket on tiket.idJenisiket = jenistiket.idJenis ORDER BY hargahr

as: alias digunakan untuk menamai kolom sesuai yang kita inginkan, order by : dugunakan untuk mengurutkan disitu order by harga, secara default kalau tidak diset ya mengurutkan terkecil ke terbesar
